% = Writes literal %, The full conversion specification must be %%
c = writes a single character, must be in single quotes to indicate it's a single character
s = writes a character string, the string has to be in double quotes to indicate it's a string

\a = Alert
\b = Backspace
\f = Formfeed 
\n = Newline (Line Feed)
\r = Carriage Return
\t = Horizontal Tab
